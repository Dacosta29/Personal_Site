Site Architecture

The application is a static website hosted on Amazon S3 and served over CloudFront for improved performance and global distribution. The infrastructure, including both S3 and CloudFront, was provisioned using Terraform, ensuring a consistent and reproducible setup.

The static site leverages S3's robust storage capabilities for high availability and durability. CloudFront, as a content delivery network (CDN), enhances the user experience by delivering the website content with low latency and high transfer speeds.

Continuous deployment is automated with GitHub Actions, streamlining updates and ensuring seamless integration of new features. The entire codebase is open-source and available on my GitHub repository.
